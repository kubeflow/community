# Projects in Kubeflow Community Distribution

The Kubeflow Community Distribution (KCD) bundles the following projects:

| Project                 | Kubeflow Subproject/Ecosystem |
| ----------------------- | ----------------------------- |
| Kubeflow Hub            | Subproject                    |
| Kubeflow Katib          | Subproject                    |
| Kubeflow Notebooks      | Subproject                    |
| Kubeflow Pipelines      | Subproject                    |
| Kubeflow SDK            | Subproject                    |
| Kubeflow Spark Operator | Subproject                    |
| Kubeflow Trainer        | Subproject                    |
| KServe                  | Ecosystem                     |

## Changes to the Included Projects

Changes to the included projects may be proposed through a Pull Request on this document by a
Kubeflow community member.

Amendments are accepted following [the KCD decision process](./charter.md#kcd-decision-process)
and MUST be documented in the [KCD decision log](./decision-log-kcd.md)

Proposals and amendments to this document are available for at least a period of one week for
comments and questions before a vote will occur.
