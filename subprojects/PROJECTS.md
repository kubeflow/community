# Kubeflow Subprojects

This document lists the existing Kubeflow subprojects grouped by their status level.

The criteria for each status level are defined in the
[maturity_requirements.md](maturity_requirements.md) document.

When a project changes its status level, this document and the
[Kubeflow installation page](https://www.kubeflow.org/docs/started/installing-kubeflow/) MUST be updated.
